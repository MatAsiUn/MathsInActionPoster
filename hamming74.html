Hamming (7,4) Code Interleaver

The Hamming (7,4) Code in Action

The Hamming (7,4) code is a [7,4,3] linear block code. It can correct a single error in each codeword. Use the text box to input a message in english characters (up to 10 characters). This message will be translated into binary and sent across a binary symmetric channel (BSC). Use the slider to select an error rate for the BSC.

Original ASCII Binary

Your message converted to standard 8-bit binary characters.

Transmitted Binary (Encoded)

The ASCII binary is split into 4-bit strings. The Hamming Code encodes each 4-bit string into a 7-bit codeword

Received Binary (After Noisy Channel)

Our message is sent across the noisy channel. Bits flipped by the channel are highlighted Red.

Errors introduced by channel: 0 bits

Decoded Binary

We then decode our codewords using nearest neighbour decoding. Successfully corrected errors are Orange. Unfixed errors (or miscorrections) remain Red.

Decoded ASCII Binary

The corrected 4-bit data chunks are extracted and reassembled into 8-bit bytes. Uncorrected bits are highlighted.

Final Uncorrected Errors: 0 bits

Reconstructed Message

Note that each character corresponds to 8 bits (2 words), both of which must be decoded correctly for the character to match.
