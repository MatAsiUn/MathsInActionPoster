Hamming (7,4) Code Interleaver

How an Interleaver Works

An interleaver protects data against burst errors, it does this by systematically systematically shuffling the sequence of our encoded symbols across multiple distinct codewords. If the channel introduces a large burst error that would cause an error if it was concentrated in 1 codeword, the interleaver ensures that it is split up over many codewords.

When a burst of errors hits the transmitted stream, the deinterleaver at the receiver reshuffles the data back to its original order.
